# BMAD-METHOD 한국어 번역 프로젝트

이 디렉토리는 BMAD-METHOD™ 프레임워크의 체계적인 한국어 번역 작업을 관리합니다.

## 📚 문서 구조

### [translation-plan-v4.md](./translation-plan-v4.md)

**번역 계획서** - 전체 번역 작업의 마스터 플랜

- 번역 현황 및 진행 상태
- 4단계(Phase) 번역 로드맵
- 파일별 상세 작업 계획
- 일정 및 체크포인트
- 번역 가이드라인 및 용어 통일 기준

### [translation-issues.md](./translation-issues.md)

**이슈 추적 문서** - 번역 중 발견되는 문제 및 개선사항

- 용어, 구조, 기술 정확성 관련 이슈
- 해결 방안 및 상태 추적

### [translation-session-log.md](./translation-session-log.md) (생성 예정)

**작업 세션 기록** - 각 번역 세션의 상세 로그

- 세션별 작업 내용
- 번역된 파일 목록
- 주요 결정사항 및 노트

## 🚀 빠른 시작

### 번역 작업 시작 전

1. [translation-plan-v4.md](./translation-plan-v4.md) 읽기
2. 현재 Phase 확인
3. 번역 가이드라인 숙지

### 번역 작업 중

1. 해당 Phase의 파일 선택
2. 번역 가이드라인 준수
3. 완료 시 계획서에 체크 표시
4. 이슈 발견 시 translation-issues.md에 기록

### 번역 작업 후

1. 품질 검토 (원문 대조, 용어 일관성)
2. 동작 테스트 (YAML 구조, 템플릿 렌더링)
3. 세션 로그 업데이트 (생성 예정)
